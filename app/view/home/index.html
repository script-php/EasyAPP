Quick Start

Here's how to create your first custom page:

  1. Create a controller: php easyapp make:controller Welcome
  2. Add a route in app/router.php: $router->get('/welcome', 'welcome');
  3. Create a view in app/view/welcome.html
  4. Visit /welcome in your browser
